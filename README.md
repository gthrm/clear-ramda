# CLEAR-RAMDA

Подробнеее про функции [ramdajs](https://ramdajs.com/) на Русском

## Что это:

Разбираюсь в ramdajs и пишу свои мысли тут

## Как запустить примеры:

`yarn install && node DIR_NAME/index.mjs`
, где `DIR_NAME` - это имя функции
